# Advanced To-Do List

- Chrome Web Store: https://chromewebstore.google.com/detail/mibfplaelemmjmeljhlfnhdmmajgdeio
- 10+ active users per week

## Project description

This is an extension for any website, with advanced functionality.

- Add a task.
- Edit a task.
- Copy a task.
- Assign a color to a task.
- Deleting tasks.
- Drag & Drop
- Data is stored in **localStorage**.

---

## Technologies used

- **React**
- **JavaScript**
- **CSS** (dimensions are given in the pixel system)
- **Vite**
